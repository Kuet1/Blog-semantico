# Blog - Projeto de Estudos em HTML/CSS
Este repositório contém o código-fonte de um blog pessoal, desenvolvido com foco na aplicação prática de HTML5 Semântico e técnicas de Layout Responsivo.

## 🎯 Objetivo do Projeto
O propósito central deste projeto foi consolidar conhecimentos fundamentais de desenvolvimento web adquiridos durante o curso de Frontend do Matheus Battisti, priorizando a estrutura semântica dos documentos (<main>, <article>, <section>, etc.) para garantir a acessibilidade e a correta hierarquia de dados, além de aprimorar a manipulação de layouts complexos com Flexbox.

## 🛠️ Desafios e Soluções Técnicas
Durante o desenvolvimento, encontrei desafios comuns na construção de interfaces que foram essenciais para o meu aprendizado:

HTML Semântico: Implementação de uma arquitetura de tags que melhora a legibilidade do código e o SEO do projeto.

Gestão de Layout com Flexbox: O maior desafio foi lidar com o redimensionamento de elementos (cards) dentro de containers flexíveis. Utilizei as propriedades flex-grow, flex-shrink e flex-basis para controlar o comportamento dos itens e evitar quebras de linha inesperadas ao limitar o crescimento dos elementos.

Para resolver o problema que eu estava tendo durante o desenvolvimento, redimencionei as imagens levando em conta que o container principal tinha **1200px**, então adicionei o *width* e *height* fixo para resolver o problema no valor de **260px** para *width* e **200px** de *height*;  


### Solução técnica extra
**Responsividade**: Ajuste de containers e elementos de mídia para garantir uma boa experiência de leitura em diferentes tamanhos de tela.

## 🚀 Tecnologias
- HTML5 (Semântico)

- CSS3 (Flexbox, Media Queries)

- Git/GitHub (Versionamento)

## 💡 Aprendizados
Este projeto foi um marco no meu início de carreira. Mais do que apenas "fazer funcionar", aprendi a pensar na estrutura do documento como um todo e a interpretar como as propriedades do CSS influenciam o fluxo do conteúdo. Foi uma excelente oportunidade para entender a diferença prática entre containers de estilo (divs) e elementos de conteúdo (article/section).
